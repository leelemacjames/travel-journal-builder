# Travel Journal Builder

A React + Vite + Tailwind scrapbook-style travel journal app.

## Features

- Builder screen to configure title, subtitle, region, page count, and photos per page
- Map + Story page layout with editable route locations
- Collage scrapbook layout with draggable photos and layer (z-index) controls
- Render mode (view-only) vs Edit mode
- LocalStorage persistence
- JSON export / import

## Getting Started

```bash
npm install
npm run dev
```

Then open the URL that Vite prints (usually http://localhost:5173).

## Deploying on Netlify

- Connect this repo to Netlify
- Build command: `npm run build`
- Publish directory: `dist`
